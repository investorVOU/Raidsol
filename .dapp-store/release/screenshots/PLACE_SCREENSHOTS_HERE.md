# Screenshots

Add 1080×1920 PNG screenshots here (portrait orientation).

Recommended shots:
1. `01-lobby.png`   — Lobby screen with wallet connected
2. `02-raid.png`    — Active raid with multiplier + risk bar
3. `03-result.png`  — Victory / extraction result screen
4. `04-pvp.png`     — PvP room lobby with players
5. `05-store.png`   — Gear store
6. `06-profile.png` — Profile / leaderboard

Max 8 screenshots. Min 2 required for Seeker submission.

## How to capture on Seeker device
- Open the app in Seeker browser
- Install as PWA (Add to Home Screen)
- Use Android screenshot shortcut (Power + Vol Down)
- Transfer to this folder and rename accordingly

Then update config.yaml screenshot URIs.
